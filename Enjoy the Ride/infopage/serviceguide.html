자전거 도로
자전거 도로 버튼을 클릭하면, 지도 상에 파란색으로 자전거 도로가 표시 됩니다.
버튼을 한번 더 클릭 시, 자전거 도로 표시가 사라지게 됩니다.
아래는 자전거 도로가 표시되지 않은 상태입니다.
자전거 수리
자전거 수리 버튼을 클릭하면, 카카오 지도의 키워드 검색을 이용하여 검색을 실시합니다.
스패너 이미지로 자전거 수리점을 표시하였고, 수리점까지 길찾기 기능을 제공합니다.
자전거 보관
자전거 보관 버튼을 클릭하면, 공공데이터 api 자전거 보관함의 정보를 받아와서
지도 상에 마커 형태로 띄워줍니다. 데이터를 많이 받아와서 약 6초~8초 정도 소요됩니다.
마우스 휠을 이용하여 줌을 축소하면 클러스터 형태로 밀집도를 보여줍니다.
시작 화면
시작 화면 버튼을 클릭하면, 웹페이지의 초기 화면으로 돌아갑니다.
길찾기 제거
길찾기 제거 버튼을 클릭하면 자동으로 목적지까지 폴리 라인이 그려집니다.
분홍색으로 그려진 선을 제거하고 싶다면, 길찾기 제거 버튼을 누르세요.
현재 위치
카카오 맵 api의 geolocation을 이용하여 현재 위치 정보를 받아옵니다.
도착 위치
마우스로 찍은 위치를 도착 위치로 설정하여 마커를 표시합니다.
길찾기 버튼을 누르면 도착 위지까지의 길찾기 기능을 제공합니다.
이때, 카카오 내비 기능을 사용합니다.
폴리 라인
자전거 수리점, 보관함, 마우스로 찍은 도착 위치 등 목적지까지
눈에 잘 보이도록 선을 그려줍니다. 카카오 맵 api의 polyline 기능을 사용하였습니다.
